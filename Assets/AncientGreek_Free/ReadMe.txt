■ Package Contents

- Meshes: 2 FBX files (Male_Elder Basic and Variant)

- Materials: 4 total
  - Standard materials: 1 for Male_Elder Basic + 1 for Male_Elder Variant
  - Shader Graph materials (customizable skin): 1 for Male_Elder Basic + 1 for Male_Elder Variant

- Prefabs: 4 total
  - Male_Elder Basic and Variant, each provided with Standard and Shader Graph CustomSkin materials
  - All configured with Humanoid rig

- Textures:
  - 2 base color textures (512x512)

- Scenes:
  - 1 demo scene showcasing all prefabs

---

■ Geometry

Character mesh triangle count: 554–600 tris.  

---

■ Customizing Skin Color (for CustomSkin Materials)

You can customize the skin color by adjusting the `Color` input node in the Shader Graph used by the material.
